# Проект: OK Pro

## Что это

Лендинг продюсерского центра / личного бренда OK (Олеся Константинова).
Одностраничный сайт с заделом под расширение в полноценный сайт + блог во второй итерации.

## Бизнес-цель

**Продажи услуг.** Сайт — это нижняя часть воронки для трафика из Instagram-таргета.

## Целевое действие

Посетитель должен:
- Заполнить форму заявки на сайте, ИЛИ
- Перейти в Telegram/WhatsApp по контактам

## Аудитория

**Гео:** Россия, СНГ.

**Основные сегменты:**
1. Женщины, неудовлетворённые внешностью, ищущие комплексное преображение (ядро beauty-марафона)
2. Начинающие специалисты/предприниматели без опыта в построении личного бренда
3. Опытные эксперты, которым нужно масштабировать узнаваемость и доход

**Главная боль:** недостаток популярности, узнаваемости, внимания.

## Услуги (5 направлений)

1. **Развитие личного бренда** — упаковка, контент-стратегия, ведение, продвижение
2. **PR и медиаприсутствие** — размещения в СМИ, работа с журналистами
3. **Beauty-преображение / Marathon (флагман)** — 2 месяца: диета, тренировки, стилист, косметология, при необходимости пластика, ежедневный психолог
4. **Видеопродакшн** — Reels/Shorts, имиджевые ролики, монтаж
5. **Стратегия продюсирования** (опционально, премиум) — комплексный пакет для медийных персон

**Цены:** на сайте не показываем. Везде CTA «Узнать стоимость» / «Оставить заявку».

## Тон и эмоция

- Обращение **на «Вы»**, дружески-тёплое
- Не корпоративно-сухое, не подростковое
- Цель первого экрана: **удивление от новизны**, ощущение «таких сайтов в нише ещё не делали»

## Анти-референсы

- Устаревшие сайты «эпохи 2015»
- Перенасыщенные блоки с большим количеством текста
- Агрессивные «инфобиз-лендинги» с криками и красным цветом

## Референсы

- WOW! Brand — agency-стиль, чистая подача
- Соломка — личный бренд, премиальный продакшн
- Личный бренд в СМИ (третий референс)

## KPI

Количество заявок в месяц.

> ⚠️ Аналитика на MVP не подключается по решению заказчика. Архитектурно предусмотреть простое подключение Метрики/GA на этапе 2.

## Источник трафика

Преимущественно платный таргет в Instagram. **Mobile-first — критично.**
