# OK Pro — документация для AI-агента

Этот набор файлов — контекст для AI-агента, разрабатывающего лендинг **OK Pro** (личный бренд / продюсерский центр Олеси Константиновой).

## Порядок чтения

Перед началом любой задачи прочитать **в этом порядке**:

1. **`AGENT_RULES.md`** — как работать, что можно/нельзя
2. **`PROJECT.md`** — что строим и зачем
3. **`STACK.md`** — технологии, версии, библиотеки
4. **`ARCHITECTURE.md`** — структура папок, конвенции кода
5. **`DESIGN_SYSTEM.md`** — токены, цвета, типографика, паттерны
6. **`CONTENT.md`** — тексты для блоков (заглушки)
7. **`ROADMAP.md`** — что делаем сейчас, что потом
8. **`DEPLOYMENT.md`** — деплой на Vercel, домен reg.ru

## Контекст одной строкой

Лендинг на Next.js 14 (App Router) + TypeScript + Tailwind, деплой на Vercel, MVP к 30 апреля, основной трафик — таргет из Instagram, мобильный приоритет.
